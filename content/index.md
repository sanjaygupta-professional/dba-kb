- [[doctoral-research-methodology]] — What makes DBA research different from applied projects
- [[emerging-digital-technologies]] — Umbrella category — technologies beyond core AI
- [[ensemble-methods]] — Combining multiple weak models (bagging/boosting) to outperform any single model

- [[logistic-regression]] — Classification algorithm mapping linear inputs to [0,1] via sigmoid
- [[transfer-learning]] — Reusing a pre-trained model's learned weights on a new task
