# Course 04 — Session 01 Slides Distillation

Opening lecture of Foundations of ML and AI (Course_04), delivered 20-Sep-2025 by [[dr-sridhar-pappu|Dr. Sridhar Pappu]] (co-founder of [[insofe|INSOFE]], acquired by [[upgrad|upGrad]]).

## Content arc

1. **Instructor bio** — 30 years spanning materials-science research, AI products, and education (Golden Gate U, Case Western, uGDX, INSOFE/upGrad).
2. **Understanding AI paradigms** — the paradigms framing. See [[concepts/ai-paradigms|AI paradigms]] for the distinctions between Symbolic AI, Machine Learning, Agentic AI, and Composite AI surfaced in the deck.
3. **Vibe-coding live demo — fraud detection app** — built in 1.5 hours for $3.19 using [[concepts/vibe-coding|vibe coding]]:
   - Tools: ChatGPT-o3, Visual Studio Code, Cline, Anthropic Claude Sonnet 4, DeepSeek
   - Data: 1000-row simulated fraud dataset (login attempts, request counts, cart value, payment method, billing/shipping country, rule flags)
   - App features: Agentic AI path vs. Composite AI path (user selectable), 70/30 train/test split, binary classification, confusion matrix, Streamlit UI + dashboard
4. **Building blocks of AI** — set-up for subsequent sessions covering linear/logistic regression, clustering, ensemble methods (visible in later Session_XX decks).
