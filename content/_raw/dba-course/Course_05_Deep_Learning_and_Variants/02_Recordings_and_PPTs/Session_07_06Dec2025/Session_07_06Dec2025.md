## Key Takeaways

**Topic:** Autoencoders, dimensionality reduction, and PCA foundations

1. **Deep Neural Networks** `[17:20]` — >2 hidden layers via parameter optimisation.
2. **Vanishing Gradient Problem** `[22:03]` — Sigmoid-derivative decay across layers.
3. **ReLU Activation Function** `[25:01]` — Linear thresholding fix.
4. **Mini-batch Gradient Descent** `[26:27]` — Small-subset update sweet spot.
5. **Epoch** `[27:38]` — One pass through training data.
6. **Learning Rate Scheduling** `[29:00]` — Step / exponential decay.
7. **Momentum** `[31:00]` — EMA of gradients smooths the path.
8. **RMSprop** `[29:54]` — LR normalised by gradient variance.
9. **Adam Optimizer** `[33:39]` — Industry-standard momentum + RMSprop combo.
10. **Parameter Initialization** `[34:37]` — Proper random init enables convergence.
11. **Regularisation** `[35:18]` — Complexity vs performance balance.
12. **Dropout Regularisation** `[36:03]` — Stochastic zeroing promotes generalisation.
13. **Batch Normalisation** `[39:49]` — Layer-input standardisation.
14. **Autoencoder** `[50:24]` — Mirror-symmetric net that reconstructs input — unsupervised.
15. **Encoding/Embedding** `[62:22]` — Compact hidden-layer vector.
16. **Dimensionality Reduction** `[64:14]` — Neural compression to lower-dim space.
17. **Principal Component Analysis (PCA)** `[103:01]` — Linear DR — uncorrelated components by variance.
18. **Covariance Matrix** `[121:10]` — Pairwise covariances — the rotation basis.
19. **Eigenvectors and Eigenvalues** `[130:12]` — Scale-only vectors + their variance magnitudes.
20. **Linear Combinations** `[140:41]` — PCs as weighted sums of original variables.
