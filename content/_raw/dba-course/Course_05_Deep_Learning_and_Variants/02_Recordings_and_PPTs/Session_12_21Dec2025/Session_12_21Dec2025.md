## Key Takeaways

**Topic:** Transfer learning and object detection in deep-learning CNNs

1. **Transfer Learning** `[28:01]` — Reuse pre-trained models; fine-tune last layers.
2. **Feature Extraction** `[35:02]` — Early layers → edges; deep layers → complex features.
3. **Pre-trained Models** `[36:07]` — ImageNet-trained nets adapted to new tasks.
4. **Fine-tuning Strategy** `[37:20]` — Unfreeze more layers as data grows.
5. **Data Augmentation** `[46:05]` — Rotation/scale synthetic samples for small datasets.
6. **Frozen Layers** `[48:53]` — Weights constant during backprop.
7. **Object Detection Problem** `[49:31]` — Locate *and* classify multiple objects.
8. **Sliding Window Approach** `[54:56]` — Brute-force multi-size scan.
9. **Region Proposal** `[56:27]` — Candidate regions → CNN classification.
10. **R-CNN** `[55:42]` — Region extraction + per-region CNN.
11. **Selective Search Algorithm** `[58:19]` — Hierarchical similar-region grouping.
12. **YOLO** `[68:29]` — Fast real-time detection.
13. **Semantic Segmentation** `[69:49]` — Per-pixel classification.
14. **Recurrent Neural Networks** `[75:06]` — Feedback-loop nets for sequences.
15. **Neural Network Memory** `[76:38]` — State carries info forward across steps.
16. **Sequential Data Processing** `[77:27]` — Context-dependent sequences.
17. **LSTM Networks** `[80:11]` — Extended RNNs for long + short-term dependencies.
