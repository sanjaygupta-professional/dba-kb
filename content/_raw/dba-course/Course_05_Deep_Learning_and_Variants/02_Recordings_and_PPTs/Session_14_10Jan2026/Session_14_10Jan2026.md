## Key Takeaways

**Topic:** Generative image models and reinforcement-learning foundations

1. **Discriminative Models** `[30:04]` — Classify or predict.
2. **Generative Models** `[30:14]` — Learn distribution to *generate*.
3. **Image Representation** `[36:30]` — Flatten image matrix → pixel-brightness vector.
4. **Feature Space Distribution** `[40:01]` — Class-specific clusters — generation target.
5. **Latent Space Sampling** `[41:32]` — Random samples → new data.
6. **Autoencoder Architecture** `[49:41]` — Encoder-decoder prior.
7. **Variational Autoencoder (VAE)** `[47:44]` — Probabilistic encoder — mean + std of latents.
8. **Probability Distribution Encoding** `[55:31]` — Gaussian parameters, not points.
9. **Sampling and Decoding** `[59:47]` — Random samples → decoder → synthetic images.
10. **Adversarial Networks (GAN)** `[71:31]` — Generator vs discriminator game.
11. **Generator Network** `[77:40]` — Noise → synthetic images.
12. **Discriminator Function** `[79:38]` — Real vs fake classifier.
13. **Adversarial Training** `[83:53]` — Alternating optimisation.
14. **Reinforcement Learning** `[109:14]` — Agent maximises cumulative reward.
15. **Agent-Environment Interaction** `[116:49]` — State → action → reward → new state.
16. **Reward Signal** `[126:28]` — Numerical feedback, positive/negative.
17. **State Representation** `[123:40]` — Relevant environment info for decisions.
18. **Value Function (Q-Function)** `[140:04]` — Expected discounted future reward.
19. **Policy Learning** `[141:43]` — Actions that maximise value.
20. **Deep Q-Learning** `[148:00]` — Neural-net approximation of value functions.
